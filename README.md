### Dependencies:

* node.js
* mongoDB
* npm

### Usage:

Call program from the command shell with a cat name:

```
// Initialize mongoDB (one terminal window)
> mongod

// Run model (different terminal window)
> node main morris
```

Cat is saved to listing of all cats.
